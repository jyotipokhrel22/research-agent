What the system does

From literature retrieval to research directions.

Priority-based retrieval

Search starts from Semantic Scholar, falls back to arXiv, and uses Tavily only when broader web retrieval is needed.

Structured extraction

Each selected paper is summarized into methods, findings, and limitations so analysis stays comparable across sources.

Gap generation

The system synthesizes repeated weaknesses across papers and turns them into concrete research gaps and suggested next directions.
